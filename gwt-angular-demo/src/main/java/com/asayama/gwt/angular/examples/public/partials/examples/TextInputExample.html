In this example, we demonstrate how to associate a text <INPUT> field with a model called "name". This model can be used elsewhere in the code. When the value of the model is updated, the view is automatically refreshed.

To see this code in action, try typing in something into the text field below. Before you enter any values into the field, the caption above the field should read "{{ GREETING }}, {{ STRANGER }}!". Once you enter a value in the field, e.g. "Jack", then you will see that caption updated to "{{ GREETING }}, Jack!" as you type in the value.

{{ GREETING }}, {{ getName()||STRANGER }}!
