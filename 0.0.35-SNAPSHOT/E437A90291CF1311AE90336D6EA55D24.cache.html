gwt-angular-prettify offers google-code-prettify right inside your GWT Angular code. Together with AngularJS's ngSantizie module, you can embed a code fragment into your screen with basic syntax coloring provided by google-code-prettify.

Below example shows a string, HTML_CONTENT, filtered through the prettify filter, which is defined in the PrettifyModule.

HTML_CONTENT is a public static final variable, which is initialized by TextResource defined in PrettifyExampleResources. Because an outright HTML would be rendered by the browser, we must escape all the mark-ups prior to injecting it into the view using the directive, data-ng-bind-html. SafeHtmlUtils.htmlEscape() is used for this purpose. 


  
  

    The following modules are required in the gwt.xml.
  

  
<inherits name="com.asayama.gwt.angular.NG" />
<inherits name="com.asayama.gwt.angular.sanitize.NGSanitize" />
<inherits name="com.asayama.gwt.angular.prettify.PrettifyModule" />


  

    The following Maven dependencies are required in the pom.xml.
  

  
<dependency>
  <groupId>com.asayama.gwt</groupId>
  <artifactId>gwt-angular-prettify</artifactId>
  <version>{{ PROJECT_VERSION }}</version>
</dependency>
<dependency>
  <groupId>com.asayama.gwt</groupId>
  <artifactId>gwt-angular-sanitize</artifactId>
  <version>{{ PROJECT_VERSION }}</version>
</dependency>
