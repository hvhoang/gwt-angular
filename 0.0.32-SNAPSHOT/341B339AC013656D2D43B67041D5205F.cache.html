Use of HttpClient is demonstrated in this example. Note that the HttpClient used in this example is based on GWT's RequestBuilder which is adapted to AngularJS framework, opposed to AngularJS's $http service object.

The HTTP GET request is sent asynchronously without blocking the user interaction. The response is a JSON object, (to be more precise, in this example, an array of JSON objects), which is parsed and loaded into an overlay type called Customers. In the HTML view, we use data-ng-repeat directive to iterate over the JSON objects in this array.
