Download via Maven

If you are looking into GWT Angular, you must have some experience with Java development. Maven is a popular artifact and dependency management tool among Java developers, and GWT Angular fits right into that paradigm. If you would rather have Maven take care of the download for you, you can start with the following configuration in your POM to get you started. gwt-angular-ng is the core API. gwt-angular-ngRoute module is useful if you want to create a single-page-application, and use hash part of your URL for site navigation. 

<dependencies>
  <dependency>
    <groupId>com.asayama.gwt</groupId>
    <artifactId>gwt-angular-ng</artifactId>
    <version>{{ GWT_ANGULAR_VERSION }}</version>
    <scope>provided</scope>
  </dependency>
  <dependency>
    <groupId>com.asayama.gwt</groupId>
    <artifactId>gwt-angular-ngRoute</artifactId>
    <version>{{ GWT_ANGULAR_VERSION }}</version>
    <scope>provided</scope>
  </dependency>
</dependencies>

You can also see the POM for the project for this web site by visiting GitHub.
