Introduction

GWT Angular brings the powerful MVC framework of AngularJS to the Java developers. By using GWT Angular, the user can take advantage of the build-time optimization of GWT together with the runtime efficiency of AngularJS.

In this tutorial, we will create a simple client-only application, using GWT Angular. The application will be client-only, meaning that even though the application is written in Java language, there is no server-side component in the final output. The resulting application can be deployed to a web server, such as Apache HTTP Server. Because the output of GWT is a set of JavaScripts, HTML, and other static resources, the entier application will run in the end user's browser. You will not need any JVM to run the final compiled output of this application (though you will need one to do the development, if you decide to follow through the steps).

Basic knowledge of Java and GWT is assumed. If you are not familiar with Java, The Java Tutorials is a good place to start. If you are not familiar with GWT, then please go over GWT Tutorial before attempting to follow this tutorial.

If you have never worked with AngularJS, that should not preclude you from being able to follow this tutorial. However, you should at be familiar with basic HTML and JavaScript programming concepts, as well as well known design patterns, such as Model View Control. I have found w3schools an excellent source of HTML and JavaScript programming, and at minimum you can skim through the Wiki page on AngularJS before you start.

In this tutorial, we will use Apache Maven to manage the project dependency, and use Eclipse for IDE. Strictly speaking, these things are not necessary, but if you plan to follow along the steps in tutorial, you may want to install and configure these tools as well.

You can find information about Maven at Apache Maven website, and you can learn about Eclipse at the Eclipese Foundation website.
